Docker Desktop + GitHub Actions + GHCR

Prototype Pipeline Template

A reusable starter kit for building, testing, packaging, and publishing your prototype web apps with Docker Desktop and GitHub Actions under the echofoxx namespace.

1. Build Locally

Use Docker Desktop and ci/local-build.ps1 to build and smoke-test the app before pushing to GitHub.

2. Publish Automatically

Push to main or create a version tag like v0.1.0. GitHub Actions builds the image and publishes it to GHCR.

3. Deploy Anywhere

Pull the image on another Docker host or lab machine and run it with docker compose up -d.

Prototype CI/CD Flow

Local Code → Docker Build → Smoke Test → GitHub Push → GHCR Image → Deploy
